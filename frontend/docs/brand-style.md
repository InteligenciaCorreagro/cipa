# Guía de estilo CIPA (verde y blanco)

## Colores de marca
- Verde principal: #1F8F4D
- Verde oscuro texto: #1F3A2F
- Verde claro fondo: #EAF7EF
- Blanco: #FFFFFF

## Uso recomendado
- Fondo general: #FFFFFF
- Texto principal: #1F3A2F
- Botones primarios: fondo #1F8F4D con texto blanco
- Bordes y contenedores: #EAF7EF o verde muy suave

## Accesibilidad (WCAG 2.1 AA)
- Texto sobre blanco usar verde oscuro (#1F3A2F)
- Evitar verde claro para texto principal
- Mantener contraste mínimo 4.5:1 en texto normal

## Tipografía
- Preferencia: Inter o sistema
- Títulos: 24–32 px, peso 600–700
- Cuerpo: 14–16 px, peso 400–500

## Componentes
- Cards con borde suave verde claro y sombra mínima
- Botones con hover sutil en verde claro
- Tablas con encabezado verde claro y texto verde oscuro
