# Manual de Usuario

## Acceso
- Inicie sesión con su usuario y contraseña.
- Si su rol es administrador, tendrá acceso a exportaciones y gestión de usuarios.

## Dashboard
- Revise el resumen de facturas registrables y no registrables.
- Verifique el estado de notas pendientes, aplicadas y no aplicadas.
- Consulte las últimas aplicaciones de notas.

## Facturas
- Registre facturas con el formulario principal.
- El monto mínimo registrable es $524.000 COP.
- El código abc123 permite hasta 5 repeticiones, siempre que la sumatoria supere $524.000 COP.
- Consulte el listado con filtros por búsqueda.

## Notas Crédito
- Registre notas con el formulario de notas crédito.
- Las notas marcadas como agente se aplican a la factura correspondiente.
- Visualice el listado por estado y saldo pendiente.
- Revise la sección de notas no aplicadas con su motivo detallado.

## Aplicación de Notas
- Use el formulario “Aplicar Nota a Factura”.
- Si una nota deja en cero la factura, la línea se elimina y la nota queda aplicada.

## Reportes
- Descargue reportes en Excel o PDF desde la sección de administración.
- Seleccione el rango de fechas y el tipo de dato a exportar.

## Estado de Notas No Aplicadas
- En el módulo de notas, consulte los motivos registrados.
- La nota se marca como no aplicada cuando corresponde.
